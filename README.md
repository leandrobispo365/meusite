# Calculadora de IMC em PHP

Este é im sistema simples desenvolvido para fins de estudo, que calcula o índice de Massa Corporal (IMC) com base no peso e altura informados.

## Tecnologias Utilizadas
* PHP
* Git & GitHub

## como fuciona 
O código recebe as variáveis de peso e altua, realiza o cálculo matemático e exibe a classificação ( Abaixo do Peso, Peso ideal, etc.) diretamente no navegador
