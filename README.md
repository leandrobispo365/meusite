## Sobre o Projeto
Este projeto foi desenvolvido para praticar conceitos básicos de PHP, HTML e lógica de programação no curso técnico de redes.

O sistema possui exemplos simples e educativos para auxiliar no aprendizado de desenvolvimento web.

---

# Funcionalidades

✅ Exibição de caixas de texto  
✅ Lista de equipamentos de informática  
✅ Tabela de alunos e notas  
✅ Uso de arrays em PHP  
✅ Estruturas de repetição  
✅ Integração entre HTML e PHP
