'# Calculadora de IMC em PHP

Este é im sistema simples desenvolvido para fins de estudo, que calcula o índice de Massa Corporal (IMC) com base no peso e altura informados.

## Tecnologias Utilizadas
* PHP
* Git & GitHub

## como fuciona 
O código recebe as variáveis de peso e altua, realiza o cálculo matemático e exibe a classificação ( Abaixo do Peso, Peso ideal, etc.) diretamente no navegador

EX: Peso: 70 kg
Altura: 1.75 m
IMC: 22.86
Classificação: Peso normal

## 📝 Sobre o Projeto

Este é um sistema de *Gestão e Registro de Matrículas* desenvolvido em PHP estruturado. O foco do projeto foi criar uma solução leve para controle de acesso, persistência de dados em arquivos locais e gerenciamento administrativo, simulando cenários reais de infraestrutura e aplicações corporativas locais.

---

## 🚀 Funcionalidades Principais

* **Painel Administrativo (Dashboard):** Área restrita para gerenciamento do sistema.
* **Controle de Acesso Seguro:** Autenticação via sessões nativas do PHP (`session_start`) com proteção de rotas para usuários logados.
* **Persistência de Dados Inteligente:** Salvamento automático de registros diretamente em arquivos planos (`matriculas.txt`).
* **Tratamento e Segurança de Inputs:** Sanitização de dados de formulário utilizando funções como `strip_tags` e `trim` para mitigar vulnerabilidades.

---

## 🛠️ Tecnologias Utilizadas

* **Backend:** PHP (Validação, gerenciamento de rotas e sessões)
* **Frontend:** HTML5 & CSS3 (Estrutura visual e painéis responsivos)
* **Ambiente de Servidor:** XAMPP (Apache) / Linux Debian para simulação de servidores corporativos
* **Controle de Versão:** Git e GitHub

---

## 💻 Como Executar a Aplicação

1. Certifique-se de ter um servidor Apache com suporte a PHP ativo (como o XAMPP).
2. Clone ou baixe este projeto para dentro do diretório raiz do seu servidor (ex: `C:\xampp\htdocs\conectaja`).
3. Abra seu navegador e acesse: `http://localhost/conectaja`.
